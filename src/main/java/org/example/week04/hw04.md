# **HW 04: HashMap Implementation and Performance Analysis**

## **Required Tasks**

### **1\. Custom HashMap Implementation**

* Implement your own HashMap implementing Map\<K,V\> interface with basic operations (put, get, remove, size,
  containsKey, entrySet, keyset, values)
* Use any collision resolution strategy of your choice

### **2\. Testing and Performance Analysis**

* Create unit tests for your HashMap implementation
* Create performance tests comparing your implementation with JDK HashMap
* Test both correctness and performance metrics

## **Bonus Tasks (Optional)**

### **⭐ Alternative Collision Resolution (1 star)**

* Implement a second HashMap using a different collision resolution strategy than your first implementation
* Add this implementation to your unit and performance tests

### **⭐⭐ Simplified ConcurrentHashMap (2 stars)**

* Implement a thread-safe HashMap using bucket-level locking
* Test concurrent access scenarios

### **⭐⭐⭐ LinkedHashMap (3 stars)**

* Implement a HashMap that preserves insertion order
* Maintain the order elements were added to the map

### **⭐⭐⭐⭐⭐ ConcurrentSkipListMap (5 stars)**

* Implement a concurrent skip list map (for the truly ambitious 😄)
* Lock-free concurrent sorted map implementation
